# Task 17: Eval Harness for the Copilot

## Whole-picture context

The role explicitly emphasizes LLM-powered product thinking. A serious implementation should include evaluation.

## Specific task goal

Create a simple evaluation framework for the AI copilot.

## Requirements

- Add a small eval dataset with 15 to 20 questions.
- Include expected answers or scoring notes.
- Measure response correctness, groundedness, tool-call success, and hallucination risk.
- Add a script or endpoint to run evals.

## Output

A lightweight but serious AI evaluation harness.

## Acceptance criteria

- An eval run produces readable results.
- Failing prompts can be identified.
- The eval framework is easy to extend.
